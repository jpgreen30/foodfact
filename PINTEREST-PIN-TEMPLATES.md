# Pinterest Pin Templates for FoodFactScanner

## General Specifications
- **Dimensions:** 1000 x 1500 px (2:3 vertical ratio)
- **Color Palette:**
  - Primary Brand: #2563eb (blue) - use sparingly for CTAs
  - Secondary Brand: #10b981 (green) - for highlights
  - Background: White or light gray
  - Text:Dark (black/dark gray)
- **Fonts:** Use bold sans-serif (Montserrat, Poppins, Open Sans)
- **Logo:** Place FoodFactScanner logo in bottom corner (optional)

## How to Use These Templates in Canva:

1. Create new design → Custom dimensions → 1000 x 1500 px
2. Set up grid/guidelines for consistent spacing
3. Create 1 master template
4. Duplicate and modify text/images for each article
5. Use brand colors #2563eb (blue) and #10b981 (green)
6. Export as PNG (high quality)

## Batch Creation Strategy:

- Create 5 base designs (one per article)
- Create 3 variations of each (different layout/colors)
- Total: 15 pins per article × 5 articles = 75 pins
- Schedule over 2-3 weeks

## Pinterest Description Template:

```
[Pin headline]

[Brief description of what reader will learn]

Link to full article: foodfactscanner.com/blog/[article-slug]

#BabyFoodSafety #Parenting #HealthyBaby #FoodFactScanner
```

## Important Notes:

- Keep text minimal (max 3-4 lines)
- Use large, readable fonts (minimum 24pt for body)
- Include clear CTA with arrow →
- Ensure mobile readability (most users on phone)
- Test pins by saving and viewing on mobile
